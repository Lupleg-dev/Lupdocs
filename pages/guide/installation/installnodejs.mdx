# Install Node.js

If you're new to using [Node.js](https://nodejs.org), you will need to install it on your development machine. Head over to [nodejs.org](http://nodejs.org) and download and install the latest version.

```
You must use Node version 18+ in order to run Lupleg.
```

```
If you've already got Node.js installed, you can skip to the next section and set up your database.
```

## Install Node Package Manager (NPM)

NPM is a huge repository of code modules that you can download and use in your projects. [Lupleg](https://lupleg.website) relies on several third-party libraries, so go ahead and [download and install NPM](https://www.npmjs.com/get-npm).
